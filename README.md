# ghana_financial_inclusion_(2017-2024)
Financial inclusion analysis using Excel, SQL, Power BI, Python &amp; data from BoG, GSS, and World Bank

##Key Insights

- Mobile account ownership (61.1%) is much higher than formal financial account ownership (37.08%).
- Deposits grew steadily from 2017 to 2022, followed by a decline of about 47% by 2024.
- Southern regions such as Greater Accra and Ashanti show higher financial inclusion than northern regions.
- Mobile money is the primary driver of financial inclusion in Ghana.

## Dashboard

This repository includes a Power BI dashboard that visualizes:

- Deposit trends (2017–2024)
- Regional mobile money usage
- Financial and mobile account ownership rates
- Total deposits vs mobile users
